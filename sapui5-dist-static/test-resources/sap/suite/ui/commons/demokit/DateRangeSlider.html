DateRangeSlider

Introduction

The Date Range Slider provides the user with a slider control for selecting a range of dates.

The overall slider range is bounded by min and max date values. In addition, each grip is bounded by the other, meaning that one grip cannot be moved over and then beyond the other grip.

The slider bar is divided into equal parts depending on the values of max, min, and totalUnits. Each part is known as a unit and represents the amount of time between two dates. Each date value may be visually marked with a tick (small vertical line on the bar) and a label that displays the date value. You may also show a bubble over each grip that displays grip date value. Overall display of labels is controlled by the stepLabels property. It is also possible to supply your own custom labels using the labels property.

The slider supports both day (default) and month granularity. This sets the minimum duration for a single unit to either day or month. For example, when month granularity is used there is one month between adjacent dates.

By default date values are formatted according to sap.ui.core.format.DateFormat medium format. You may use the setDateFormat() method to specify a custom format. When month granularity is used then only month and year are displayed.

The slider can be controlled via mouse or keyboard. The liveChange event is fired as the user slides the grip along the bar. The change event is fired when a grip is set to its final value (mouse button released).

Examples

Basic DateRangeSlider

DateRangeSlider without Bubbles

You can make the date range slider bubbles not display using the showBubbles property.

DateRangeSlider with Custom Range and Grip Positions

You can set the maximum and minimum values for the slider using the setMin() and setMax() functions. By default the slider uses a range of 365 days, where the minimum date is one year before the current date. This can be changed to increase or decrease the range of dates by setting the desired minimum and maximum values. You can also change the position of the slider grips using setValue() and setValue2().

DateRangeSlider with Custom Labels

You can create your own labels and display values other than dates. Here is a slider that displays labels for year quarters.

DateRangeSlider with Month Granularity

The default granularity for DateRangeSlider is DAY. You can change the Granularity to MONTH using the setMonthGranularity() function. It can again be switch to DAY using the setDayGranularity() function.

Locale Specific Date Formatting

This example shows how to override the default date format with your own locale specific formatting. This reformats labels, bubble text labels and tool tips for the control.

DateRangeSlider with Pinned Grip

This example shows how to pin a grip to a specific date such that it cannot be adjusted. Here the left grip is pinned while the right grip can still be adjusted. Once a grip is pinned it cannot be moved by dragging, clicking the slider bar, or using the keyboard arrow keys.

Keyboard Support

It is possible to move the slider grips using the arrow keys. Arrow left and Arrow down will decrease the grip value. Arrow right and Arrow up will increase the grip value.

ARIA Support

The DateRangeSlider control implements the following properties from the ARIA role slider:

  • aria-valuemax
  • aria-valuemin
  • aria-valuenow
